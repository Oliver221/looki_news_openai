# Looki News TV Wall

为小米电视全屏展示设计的 Looki / Looki L1 动态资讯墙：
- 动态聚合海外/国内新闻（Google News RSS）
- 动态聚合用户评论（Reddit + Hacker News）
- 正向内容直接展示
- 负向内容附带 AI 自动改进建议
- 温暖风格、滚动 feed 流，适合办公室屏保替换

## 我如何看到这个页面（最快）

> 本项目当前是 **无外部依赖** 的 Node 版本，不需要先 `npm install`。

1. 进入项目目录

```bash
cd /workspace/looki_news_openai
```

2. 启动服务

```bash
node server.js
```

3. 在电脑浏览器打开

```text
http://localhost:3000
```

4. 电视访问方式（二选一）
- **同一台设备直连电视浏览器**：电视浏览器访问运行该服务的机器 IP，例如 `http://192.168.1.23:3000`
- **HDMI 投屏**：电脑打开页面后全屏（F11）输出到电视

## 使用 npm 启动（可选）

```bash
npm start
```

## 数据接口

- `GET /api/feed`
  - 返回聚合后条目、统计信息（新闻数、评论数、负向数、更新时间）

## 常见问题

- 页面打不开：确认 `node server.js` 正在运行，且防火墙放行 3000 端口。
- 只有示例数据：说明外部源暂时不可达，系统会自动展示 fallback，网络恢复后会自动切回实时数据。
